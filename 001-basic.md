## jsの文法と異なる4つのルール
1. var 不要
2. ;   不要
3. {}  インデントで表現
4. ()  曖昧性がない場合は不要

jsの場合、
```
var score = 82;
if (score > 80) {
  alert("OK");
}
```

CoffeeScriptの場合、
```
score = 82
if score > 80
  alert "OK"
```
