# Array

# 配列の宣言
jsとおなじようにこのようにもできるし
```
a = [1, 2, 3]
```

CoffeeScriptでは改行を使って配列の宣言ができる
```
b = [
  1
  2
  3
]
```

範囲記号を使った宣言もできる
```
m = [0..5] // => [0, 1, 2, 3, 4, 5]
m = [0...5] // => [0, 1, 2, 3, 4]
```

# 配列の範囲選択
範囲記号を使って配列の範囲取得もできる
```
m = [0..5]
m[1..3] // => [1, 2, 3]
m[..] // => [0, 1, 2, 3, 4, 5]
m[2..] // => [2, 3, 4, 5]
m[..2] // => [0, 1, 2]
m[1..-2] // => [1, 2, 3, 4]
```

# 配列の部分代入
すごい使いたくないけど、こんなんも紹介されてた
```
m = [0..5]
m[..2] = ["a", "b", "c"]
m // => ["a", "b", "c", 3, 4, 5]
```
